synthpy.client.generate.GenerateClient.get_documents¶

GenerateClient.get_documents(*args, **kwargs)¶

Generate one or more synthetic documents.

Parameters

  • collection (str, optional) – the collection we want to generate documents for. If not set, we will generate documents for all collections belonging to the namespace.

  • size (int, defaults to 1) – the target number of documents to generate.

  • namespace (str, optional) – the namespace we should generate documents for. Can be omitted if previously set when initializing the client.

Warning

There is no guarantee you will get exactly size documents in the output. This is because some constraints (such as primary key/foreign key constraints) are incompatible with that exactness. You are guaranteed, however, to get at least that number of documents.
