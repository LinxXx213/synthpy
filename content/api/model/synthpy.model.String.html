synthpy.model.String¶

class synthpy.model.String(variant)[source]¶

Model type for a JSON string

The variant argument must be one of:

  • str: a valid regex string to sample from

  • Faker: the specification of a faker generator to sample from

  • DateTime: the specification of a date/time/datetime range to sample from

  • Categorical: the specification of a categorical domain to sample from. Elements must be strings.

Parameters

variant (optional) – a variant of string model, as described above.
