synthpy.model.Bool¶

class synthpy.model.Bool(variant)[source]¶

Model type for a JSON boolean

The variant argument must be one of:

  • float: a float between 0. and 1. encoding the probability for the boolean to be True.

  • bool: specifies that the Bool model is constant at the given value.

  • Categorical: the specification of a categorical domain to sample from. Elements must be booleans.
