synthpy.client.override.OverrideClient.optionalise¶

OverrideClient.optionalise(*args, **kwargs)¶

Make a field node optional (not required)

Exactly the same as OverrideClient.set_optional(field, optional=True).

Parameters

  • field (str) – the FieldRef of the field node to optionalise.

  • namespace (str, optional) – Can be omitted if previously set when initializing the client.
