synthpy.client.override.OverrideClient¶

class synthpy.client.override.OverrideClient(client)[source]¶

Base class for the Namespace API.

Note

Do not construct this class directly. Access it from the root Synth client instead.

Example

>>> from synthpy import Synth
>>> client = Synth()
>>> client.delete_override("my_collection.content.a_field")
