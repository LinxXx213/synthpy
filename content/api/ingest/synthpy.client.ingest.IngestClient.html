synthpy.client.ingest.IngestClient¶

class synthpy.client.ingest.IngestClient(client)[source]¶

Base class for the Ingest API.

Note

Do not construct this class directly. Access it from the root Synth client instead.

Example

>>> from synthpy import Synth
>>> client = Synth()
>>> client.put_documents(namespace="my_namespace", collection="my_collection", document={"yes?": True})
