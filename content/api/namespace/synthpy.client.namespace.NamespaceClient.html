synthpy.client.namespace.NamespaceClient¶

class synthpy.client.namespace.NamespaceClient(client)[source]¶

Base class for the Namespace API.

Note

Do not construct this class directly. Access it from the root Synth client instead.

Example

>>> from synthpy import Synth
>>> client = Synth()
>>> client.get_namespaces()
